# **Final Game**

This folder contains the code for the final version of the game.

### Components

Arduino Uno - The hardware necessary to run the code and connect all the devices together
Adafruit Neopixel LED Strip - A strip of LEDs with a library made specifically for using it
LCD I2C - A display that can be used to show up to 2 lines of text
Rapid 66mm 9 ohm Speaker - A speaker that can be used to play audio, usually basic tones unless other libraries are used
Push button - a button that completes a circuit and allows a signal to go through when pressed
